## 项目创意摘要

千乘坊是一个“出租车司机分销 + 本地生活商城 + 网约车派单”的 O2O 闭环平台。统一司机账户同时具备接单、分销和司机专区消费能力；返现和返现券只允许站内循环，不支持提现；首版在武汉单城跑通商城与出行双主线 MVP。
