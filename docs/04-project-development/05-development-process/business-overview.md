**最后更新：** 2026-03-29  

## 1. 项目概述

千乘坊是一套“出租车司机分销 + 本地生活商城 + 网约车派单”的 O2O 闭环系统。
司机既是接单司机，也是平台分销商，同时还是司机专区消费者。乘客可通过司机专属二维码进入小程序购买商品或叫车；司机在乘客支付完成后获得平台内返佣，返佣仅可在平台生态内继续消费，形成可持续的站内循环。

## 2. 首版建设目标

- 在武汉单城跑通商城与出行双主线
- 跑通“扫码归因 -> 下单支付 -> 返佣入账 -> 司机专区复购”闭环
- 跑通“乘客叫车 -> 调度派单 -> 司机接单 -> 行程完成 -> 支付结算”闭环
- 形成统一司机账户体系和基础运营后台

## 3. 首版交付范围

### 3.1 用户端与后台

- 乘客微信小程序
- 司机轻端微信小程序
- 司机重端 Android App
- 运营 Web 后台

### 3.2 后端服务

- `gateway-api`
- `platform-core`
- `dispatch-engine`

### 3.3 业务模块

- 统一账户与司机准入
- 商城、商品、二维码与归因
- 商城订单、支付承接与返佣
- 司机钱包、返现券与司机专区
- 出行订单、计价主链路与调度
- 消息、工单、审计与基础风控
