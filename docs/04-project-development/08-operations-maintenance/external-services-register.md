# 上线外部服务清单

**项目名称：** 千乘坊（ride-loop）  
**文档状态：** 草稿  
**负责人：** AI 软件工厂  
**主要读者：** 项目负责人 | 商务 | 运维 | 架构  
**上游输入：** 系统架构 | 技术选型 | 部署说明 | 报价任务详单  
**下游输出：** 第三方服务准备清单 | 上线前资质与账号准备清单  
**最后更新：** 2026-03-29  

## 1. 文档边界

- 本文档列出第一版上线需要或建议准备的外部服务。
- 本文档只列服务内容、用途、是否必需、申请主体和典型计费方式。
- **第三方服务申请工作、资质办理工作、实际充值和服务费用均不计入本次开发报价，也不计入开发任务详单。**

## 2. 使用说明

- “必需”表示首版上线无法绕开。
- “建议”表示商业上线强烈建议准备，但若采用临时替代方案，MVP 阶段可能短期可延后。
- “可选”表示不是首版上线阻塞项，但中后期很可能需要。
- 具体价格以签约时官方报价、合同价格或渠道折扣为准，本文件不引用实时价格。

## 4. 建议按阶段准备

### 4.1 开发联调前必须准备

- 小程序 `AppID`
- 微信支付商户号
- 地图 Web 服务 Key
- Android 地图/定位能力
- 对象存储
- PostgreSQL / Redis 运行资源

### 4.2 进入测试环境前建议准备

- Android Push 通道
- 短信验证码服务
- 域名、DNS、HTTPS 证书
- 日志与告警平台

### 4.3 商业上线前强烈建议准备

- 隐私号/号码保护
- OCR 识别服务
- CDN
- 性能监控 / APM

## 5. 不纳入开发报价的内容说明

- 第三方商务洽谈与签约
- 主体认证、资质申请、备案、白名单申请
- 平台保证金、预充值和套餐采购
- 云资源月租、年租或按量费用
- 正式生产环境运维值守费用

## 6. 与开发任务的关系

- 上述外部服务会影响最终上线时间，但不计入开发任务详单的人天报价。
- 若因第三方服务未准备好导致联调或上线延期，通常应作为外部阻塞项单独记录。
- 报价任务详单仅覆盖系统研发、联调、测试和上线准备动作；详见 [开发工作量评估详单](../05-development-process/quotation-task-breakdown.md)。
